分支管理
-----
### 一、基本命令
```
查看分支: git branch
```
```
查看远程分支: git branch -r
```
```
查看所有分支: git branch -a
```
```
创建分支: git branch <name>
```
```
切换分支: git checkout <name>
```
```
创建+切换分支: git branch -b <name>
```
```
合并分支到当前分支: git merge <name>
```
```
删除分支: git branch -d <name>
```
```
强制删除分支: git branch -D <name>
```
```
查看分支跟踪情况: git branch -vv
```
